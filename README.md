# MagicDate
The program is intended to tell the user if the month multiplied by the day is equal to the last two digits of the year.
The program will prompt user for a month, enter month in max two digit format and hit enter. Next user will be prompted for a day of the month, enter day in max two digit format and hit enter. Lastly, the user will be prompted to enter a year in two digit format. The result will tell you if the date entered is a Magic Date or not.
